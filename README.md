## Local Setup

1. Clone the repo

```bash
git clone https://github.com/kushal-khare-official/mercor-cash-app.git
```

2. run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
